Custom CSS is the path to inconsistent UI

What’s the issue?

Custom CSS

.description {
  margin-bottom: 20px;
  font-size: 16px;
}

.count {
  margin-left: 0.75em;
  font-size: 14px;
  opacity: 0.5;
}

😭

CSS is too powerful

}
